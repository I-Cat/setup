# Install OpenVPN
I recommend to use [https://github.com/Nyr/openvpn-install](https://github.com/Nyr/openvpn-install)

Install OpenVPN using this script:
```
wget https://git.io/vpn -O openvpn-install.sh && bash openvpn-install.sh
```
